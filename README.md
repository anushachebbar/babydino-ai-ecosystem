# BabyDino AI Ecosystem

A React-based business review website developed as part of an internship assignment. The project analyzes Zenith India's AI ecosystem by reviewing its products, business value, strengths, and areas for improvement.

## Features

- Responsive React website
- Overview of Zenith India's AI ecosystem
- Product analysis of:
  - Zenith
  - J.E.R.K.
  - MusicLabelAI
  - AIInsight
  - Ai4Education
- Business recommendations and suggested improvements
- Product comparison table
- Clean and modern user interface

## Technologies Used

- React.js
- Vite
- JavaScript
- CSS
